# Construction Field Management App

A modern mobile-first construction management solution designed to streamline site operations, workforce management, attendance tracking, material monitoring, invoicing, and project reporting.

Built using React Native, Expo, TypeScript, and Redux Toolkit, the application helps project managers, engineers, supervisors, and site teams efficiently manage construction projects from a single platform.

---

## Features

### Workforce Management

* Employee tracking
* Role-based access
* Worker allocation
* Team management

### Attendance Management

* Daily attendance recording
* Workforce monitoring
* Attendance reports

### Daily Progress Reports (DPR)

* Site progress tracking
* Daily work updates
* Progress documentation
* Project status monitoring

### Material Management

* Material inventory tracking
* Material consumption monitoring
* Stock management

### Invoicing

* Invoice generation
* Billing management
* Payment tracking

### Analytics Dashboard

* Project performance insights
* Workforce analytics

### Project Monitoring

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/construction-field-management-app.git
```

Navigate to the project:

```bash
cd construction-field-management-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Launch using:

* Android Emulator
* iOS Simulator
* Expo Go
